# Rules

- Follow DRY Principle - Clean code and clear components
- `docker-compose.yml` file will control the hosing of this entire project
- Leave proper comments and double new line characters to create section wide separation
- Dashboards must be simple, clean, minimal, and production-quality with a Notion-like gray/black palette, fully responsive, and designed using the `frontend-design` skill.
- Both code will run in separate docker container
